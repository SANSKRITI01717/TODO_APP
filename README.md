# TODO_APP
Todo app with edit, delete, and task completion functionality.

# Todo App

# Description

This is a simple Todo App that allows users to add, edit, delete, and manage tasks efficiently. It includes interactive buttons and checkbox functionality to control task states and improve usability.

#Features


Add new tasks
Edit existing tasks
Delete tasks
Mark tasks as completed using a checkbox
Automatically disable the edit button when a task is marked completed
Keep the Delete button active even when a task is checked
Smooth and user-friendly task management

# Application Behavior


Clicking the Add button creates a new task
Clicking the Edit button allows updating a task
Clicking the Delete button removes a task
When the Checkbox is selected:
The task is marked as completed
The Add button becomes disabled
The Delete button continues to work
